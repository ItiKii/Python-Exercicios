# 🧠 Lógica de Programação em Python

Esta pasta contém exercícios focados em **lógica de programação**.

A lógica é a base da programação e permite resolver problemas de forma estruturada.

---

# 📚 Conceitos praticados

- if
- elif
- else
- loops
- validação de entrada do usuário
- estruturas de repetição

---

# 📄 Arquivos deste diretório

## validacao_input.py

Script que demonstra como validar dados digitados pelo usuário.

Exemplos:

- verificar se é número
- verificar se é texto
- impedir entradas inválidas

---

## while_exemplos.py

Exemplos de uso do loop `while`.

Treina:

- repetição de código
- validação contínua de dados
- controle de fluxo

---

## exercicios_condicionais.py

Exercícios utilizando:

- if
- elif
- else

Treinando tomada de decisões no código.

---

# 🎯 Objetivo

Fortalecer a base de lógica de programação para resolver problemas com Python.
