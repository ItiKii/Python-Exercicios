# 📂 Leitura de Arquivos em Python

Esta pasta contém exercícios focados em **leitura de arquivos utilizando Python**.

Aprender a trabalhar com arquivos é essencial para automação, manipulação de dados e desenvolvimento de scripts.

---

# 📚 Conceitos praticados

- open()
- leitura de arquivos
- read()
- readlines()
- loops com arquivos
- strip()
