# 📊 Manipulação de Dados em Python

Esta pasta contém exercícios focados em manipulação de dados utilizando Python.

Esses conceitos são muito utilizados em:

- automação
- análise de dados
- scripts de processamento de informações

---

# 📚 Conceitos praticados

- listas
- strings
- split()
- processamento de dados
- loops

---

# 📄 Arquivos deste diretório

## exemplo_split.py

Exemplo de uso do método `split()` para dividir textos em partes.

Muito utilizado para processar dados.

---

## exemplo_listas.py

Exemplos de manipulação de listas.

Treina:

- criação de listas
- acesso a elementos
- loops com listas

---

## exercicio_processamento.py

Exercício de processamento de dados.

Treina:

- leitura de dados
- separação de informações
- manipulação de strings

---

# 🎯 Objetivo

Aprender a transformar dados brutos em informações manipuláveis utilizando Python.
